IANS DEV: style guide

Our design language

Consistent design communicates that our content is high quality and trustworthy. It can help guide learners through content.

This Style guides is a living document that will be updated to reflect current content needs. It provides the basic building blocks to create somethig unique!

Grid

The IANS layout grid uses the Foundation grid. It defaults to a 12 column, nestable grid that will adapt to different screen sizes.

The examples in this section shows different basic features of the grid to help you lay out your pages.

Please refer to the Foundation documentation for more advanced features: http://foundation.zurb.com/sites/docs/grid.html

Colum-based layout

By selecting the number of cells to merge in each row, we can create many different layouts such as the two examples below.

Each grid cell below will occupy 12 cells out of 12 on small devices (thereby taking up all the row width). It will take up 6 on medium devices and 3 on large devices.
